Obliq

Remote working culture is inevitable. Doesn't matter if you like it or not. And one of the most important component of remote working is asynchronous communications.

Screen recording becomes really crucial to the daily workflows of remote work. Be it feature showcase, product demos, new employee on-boarding etc. It avoids back and forth emails, long (and unreadable documents), confusing text messages.

But there are a lot of screen recording apps in the market, why did I build another?

Most of the popular screen recording apps are based on web technologies (Electron). Electron is good as it allows you to run same piece of code on Mac, Windows and Linux. But it comes with its own disadvantages
